# High School Hangout - Team Standings

This directory contains HTML files for publishing High School sports standings from The Frederick News-Post.

## Files

- `index.html` - Main landing page with links to all sports
- `standings_football_2025_10_23.html` - Football standings
- `standings_girls_flag_football_2025_10_23.html` - Girls Flag Football standings
- `standings_boys_soccer_2025_10_23.html` - Boys Soccer standings
- `standings_girls_soccer_2025_10_23.html` - Girls Soccer standings
- `standings_volleyball_2025_10_23.html` - Volleyball standings
- `standings_field_hockey_2025_10_23.html` - Field Hockey standings

## Data Source

All data is extracted from The Frederick News-Post High School Hangout page, updated through October 21, 2025.

## GitHub Pages

This directory is configured to be served via GitHub Pages. Visit the published site at:
`https://newsappsUMD.github.io/frederick_sports/`

## Updates

To update the standings:
1. Extract new data from the latest PDF
2. Update the HTML files in the `hs_hangout` folder
3. Copy updated files to this `docs` folder
4. Commit and push changes

The site will automatically update when changes are pushed to the main branch.
